# Store Listing Drafts

These files are working drafts for App Store Connect and Google Play Console. They are not generated into the app.

- `app-store.md` — iOS listing, privacy, and review notes.
- `play-store.md` — Android listing, Data Safety, and review notes.

Public/VPS OpenCode servers should use HTTPS. Local/private and VPN/Tailscale OpenCode servers may use HTTP.
